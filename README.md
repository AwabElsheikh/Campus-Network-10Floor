# 10-Floor Campus Network Infrastructure
Built in Cisco Packet Tracer 9.0 as part of Information Systems coursework.

## What's included
- Full 10-floor hierarchical network topology
- Router-on-a-stick with VLANs for inter-floor routing
- DHCP auto IP assignment for all devices
- Mixed wired/wireless devices per floor
- Budget report (Rp 820M estimated real-world cost)

## Tools used
- Cisco Packet Tracer 9.0
- 2911 Router, 2960 Switches
- VLANs 101–110 per floor
