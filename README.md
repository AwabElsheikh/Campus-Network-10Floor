# 10-Floor Campus Network Infrastructure

A complete enterprise campus network designed and simulated using **Cisco Packet Tracer 9.0** as part of an Information Systems networking project.

---

## Project Overview

This project demonstrates the design and implementation of a scalable 10-floor campus network following the hierarchical network model. The network provides secure communication between departments through VLAN segmentation, Router-on-a-Stick inter-VLAN routing, DHCP services, and wired/wireless connectivity.

---

## Features

* 10-floor campus topology
* Hierarchical network design
* VLAN segmentation (VLAN 101–110)
* Router-on-a-Stick (Inter-VLAN Routing)
* DHCP automatic IP assignment
* Wired and wireless devices
* Trunk links between switches
* Budget estimation
* Professional project documentation

---

## Technologies Used

* Cisco Packet Tracer 9.0
* Cisco 2911 Router
* Cisco Catalyst 2960 Switches
* DHCP
* VLANs
* IEEE 802.1Q Trunking
* Wireless Access Points
